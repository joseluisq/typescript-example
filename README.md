# Typescript Example

> Small example of how to run [Typescript](https://www.typescriptlang.org/) code.

## Install

1. Clone repository

```sh
git clone --depth 1 https://github.com/joseluisq/typescript-example.git
```

2. Install dependencies

```sh
yarn
```

## Usage

1. Run

```sh
yarn start
```

2. Test

```sh
yarn test
```

5. Build

```sh
yarn build
```
